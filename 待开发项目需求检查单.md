# 待开发项目需求检查单

## 1. 新手引导界面

**描述**: 在用户第一次打开软件时显示引导界面，帮助用户了解软件的基本功能。

**需求**:
- [ ] 首次启动软件时自动显示新手引导界面
- [ ] 引导界面只显示一次，后续启动不再自动显示
- [ ] 在设置中添加选项，允许用户手动重新显示新手引导

**实现提示**:
- 使用 UserDefaults 存储新手引导是否已显示的状态
- 首次启动时检查状态，如未显示则显示引导界面
- 设置中添加"重新显示新手引导"按钮

---

## 2. 移动媒体文件功能

**描述**: 支持通过选中/框选媒体文件，并将选中的文件移动到目标文件夹。

**需求**:
- [ ] 单击选中媒体文件，被选中的文件显示蓝色边框高亮
- [ ] 再次单击已选中文件，取消选中状态
- [ ] 支持拖动鼠标框选多个文件（拖动时形成矩形选框）
- [ ] 拖动选中的媒体文件到侧边栏中的文件夹，实现文件移动（注意：是移动，不是复制）

**实现提示**:
- 维护一个已选中文件的集合/状态
- 在 MediaCell 或 MediaGridView 中添加蓝色边框样式
- 实现拖拽框选逻辑（记录鼠标按下和抬起的位置，形成矩形区域）
- 实现拖拽操作，将选中的文件移动到目标文件夹（使用 FileManager 实现文件移动）

---

## 3. 查看系统相册功能

**描述**: 支持打开和浏览 macOS 系统相册（.photoslibrary 格式），并能查看其中的 Live Photo。

**需求**:
- [ ] 识别并支持打开 .photoslibrary 格式的文件夹
- [ ] 能查看相册中的照片列表
- [ ] 能播放/查看相册中的 Live Photo（动态照片）

**实现提示**:
- 在 FolderNode 或扫描逻辑中添加对 .photoslibrary 后缀的识别
- 使用 Photos.framework 或直接读取 .photoslibrary 数据库（SQLite）
- Live Photo 包含静态图片和动态视频部分，需要同时加载并播放
