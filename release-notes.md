# PhotoView 1.5.9 更新说明

## 新增功能
- 移动媒体文件功能：支持选中媒体文件并拖拽到侧边栏文件夹移动
- 全选按钮：选中当前显示的所有媒体文件
- 取消选择按钮：清除所有选中状态
- Shift+点击：选中连续范围的媒体文件
- 列表模式支持拖拽移动

## 问题修复
- 修复单击选中没有蓝色边框提示的问题
- 列表模式下支持拖拽移动

## 中英文界面优化
- 全选和取消选择按钮支持中英文

# PhotoView 1.5.9 Release Notes

## New Features
- Move media files: Select media files and drag to sidebar folders to move
- Select All button: Select all visible media files
- Deselect button: Clear all selections
- Shift+click: Select range of media files
- List view supports drag and move

## Bug Fixes
- Fixed blue border not showing when clicking to select
- List view supports drag and move

## i18n Optimization
- Select All and Deselect buttons support Chinese/English
